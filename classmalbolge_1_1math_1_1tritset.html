Detailed Description

template<std::size_t N, typename T = trit::min_width_type_t<N>>
class malbolge::math::tritset< N, T >

Simple ternary bitset equivalent.

Allows trit manipulation of a ternary value.

Template Parameters
NNumber of trits in set
TUnderlying integer storage type, defaults to trit::min_width_type_t<N>
