Detailed Description

Exposes a C89 interface to the important parts of the library.

A C interface makes accessing functionality from WASM or creating bindings to other languages much easier. It is not threadsafe.

Asynchronsouly passes buffer to vcpu to use as user input.

buffer is copied into the vCPU, so this can be called before vcpu is ready to receive user input. As the data is copied, you need to free buffer manually (assuming it was malloc-ed).

If the program was in a MALBOLGE_VCPU_WAITING_FOR_INPUT state, this will resume it.
