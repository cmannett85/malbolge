Detailed Description

A wrapper around a virtual_cpu that allows a function sequence to be executed on it.

This class can be moved but not copied.

void malbolge::debugger::script::script_runner::run ( virtual_memory  vmem,
const functions::sequence &  fn_seq 
)

Runs fn_seq on an internal vCPU.

This is a blocking call. The internal vCPU's lifetime is the same as this function's, this means that the script_runner instance can be reused.

Parameters
vmemInitialised virtual memory
fn_seqFunction sequence
Exceptions
base_exceptionThrown if fn_seq is malformed, or the program execution fails
