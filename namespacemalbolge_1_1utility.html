A function that extracts the next character from an input stream.

This function exists to hide the complexity of extracting a character from browser Javascript in Emscripten, this function needs to block the WASM code but not block the JS runtime in the browser (this is achieved by Asyncify).

Parameters
strInput stream
cExtract character
blockIf true, will block on the stream until there is data to read or an error occurs. If false, will return immediately
