Detailed Description

Allows one thread to control flow of execution in another.

The controlling thread uses open(std::size_t) and close() to free/block the controlled thread (a gate instance is shared between them). The controlled thread uses operator()() to notify the controller when execution passes through the gate.

Behaviour is undefined is the number of controlled threads is greater than one.

This class can be moved, but not copied.

using malbolge::utility::gate::notifier_type = std::function<void (bool closed)>

A callback type that can be used to notify when a gate is closed and opened.

This is called just before the controlled thread blocks, and again when unblocked. It is not called if the gate is open.

Parameters
closedTrue when the gate is closed and will block, false when re-opened
