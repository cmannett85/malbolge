Macro to create a string_constant from a literal.

Due to limitations in C++ we cannot use a user-defined string literal function to create a type based upon a parameter pack of chars (e.g. "hello"_str). So the simplest approach is to use the preprocessor to write out the template char arguments for us.

There is no requirement to use this, it just makes schema definitions a little easier to read.

Note
There is a limit of 128 characters
