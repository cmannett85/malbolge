template<typename Name, typename... Args>
class malbolge::debugger::script::function< Name, Args >

Represents a script function.

Argument names must be unique, it is a compilation failure otherwise.

template<typename Name , typename... Args>
template<typename... ConArgs>
constexpr malbolge::debugger::script::function< Name, Args >::function ( ConArgs...  cargs)
inlineconstexprnoexcept

Constructor.

Each constructor argument can either be:

  • A function_argument specialisation, in which case it can appear anywhere in the sequence as the name will be used to assign it
  • A type implicitly convertible to a function_argument, in which case is has to appear in the order the function_arguments are specified Any remaining unassigned arguments are assigned their default value. It is a compliation failure if there are more cargs than function_arguments.

It is a compilation failure if an unknown function_argument type is provided.
