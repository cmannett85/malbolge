Detailed Description

Wraps a mutex type following the Mutex concept, in a generic type.

This wrapper allows functions to receive generic mutex typees without being templated themselves.

It also allows for empty initialisation to mimic a dummy mutex (i.e. a mutex type that doesn't actually do anything.
