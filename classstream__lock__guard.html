Detailed Description

template<typename Mutex>
class stream_lock_guard< Mutex >

A std::lock_guard clone that only acquires the lock if stream is not in the standard library.

std::cout, std::cerr, std::clog, and std::cin are threadsafe - but std::iostream is not. This lock guard will not acquire the lock if the stream is a standard one.

This class can be moved, but not copied.
